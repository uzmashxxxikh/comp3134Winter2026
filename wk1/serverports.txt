What is the definition of server port?
A server port is a logical communication endpoint on a host that identifies a specific network service or process so that data can be delivered to the correct application.
​

What is server port range?
TCP and UDP port numbers range from 0 to 65535, giving 65,536 possible ports.
​

How many bits does number above represent?
The range 0–65535 comes from an unsigned 16‑bit field in TCP/UDP headers, so it represents 16 bits.
